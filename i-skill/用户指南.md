# i-skill 用户指南

欢迎使用 i-skill！本指南将帮助您了解如何使用 i-skill 获得个性化响应。

## 什么是 i-skill？

i-skill 是您的个人档案生成器，它通过分析您的对话来了解您，并帮助 AI 助手提供个性化响应。

### 工作原理

i-skill 分析您的对话以了解：
- 您的兴趣（如：科技、体育、工作）
- 您的沟通风格（如：简洁、详细）
- 您的偏好（如：喜欢代码示例、偏好简单解释）
- 您在不同话题上的专业水平

基于这些信息，i-skill 创建个人档案，帮助 AI 助手根据您的需求定制响应。

## 如何使用 i-skill

### 方式一：手动激活（简单）

**只需说："激活个性化"**

就这样！i-skill 将开始将您的档案应用于所有响应。

**激活前，i-skill 需要了解您：**
- 与 AI 进行 3-5 次对话
- 谈论您感兴趣的话题
- 自然地分享您的偏好

**激活后：**
- i-skill 自动个性化您的所有响应
- 无需再次激活
- 您的档案会持续学习和改进

### 方式二：自动激活（无需操作）

许多技能可以自动激活 i-skill，以提供更好的个性化响应。

**工作原理：**
1. 当您使用依赖 i-skill 的技能时
   ```yaml
    depends:
   - i-skill
   ```yaml
2. i-skill 在后台自动激活
3. 您的个人档案被加载并应用
4. 您无需任何额外努力即可获得个性化响应

**示例：**
- 您使用"编程助手"技能
- 该技能声明依赖 i-skill
- i-skill 自动激活
- 编程助手现在了解您的编程风格和偏好
- 您获得个性化的代码建议和解释

**优势：**
- 无需手动激活
- 档案在相关时自动应用
- 跨不同技能的无缝体验

### 为您的技能添加 i-skill 支持

如果您是技能开发者，想让您的技能支持个性化，只需在技能的 SKILL.md 中添加依赖声明：

**步骤：**
1. 在您的技能 SKILL.md 中添加 `depends` 字段
2. 声明对 i-skill 的依赖
3. 添加依赖管理说明

**示例：**
```yaml
---
name: "YourSkill"
description: "Your skill description"
depends:
  - i-skill
---

## Dependency Management

This skill depends on `i-skill` for personalized responses.

**When Invoked**:
- Automatically activate i-skill
- i-skill will load user profile and provide personalized context
- See i-skill's USER_GUIDE.md for detailed personalization features
```

**就这么简单！** 添加这几行后，当您的技能被调用时，i-skill 会自动激活并提供个性化上下文。

## 管理您的档案

### 查看您的档案
说以下任意内容：
- "查看我的档案"
- "显示我的档案"
- "我的档案"
- "view my profile"

这将显示 i-skill 对您的了解。

### 暂停个性化
如果您想暂时停止个性化：
- 说："暂停个性化" 或 "停止个性化"
- i-skill 将停止应用您的档案
- 您的档案仍然保存，可以稍后恢复

### 恢复个性化
要再次开始个性化：
- 说："恢复个性化" 或 "继续个性化"
- i-skill 将再次开始应用您的档案

### 重置您的档案
如果您想重新开始：
- 说："重置档案" 或 "删除档案"
- i-skill 将要求确认
- 确认后，您的档案将被删除
- i-skill 将从头开始了解您

### 直接编辑您的档案
您也可以直接编辑档案文件：
- 档案位置：`./user_data/myself.md`
- 在任何文本编辑器中打开文件
- 根据需要进行更改
- 保存文件
- i-skill 将使用您更新的档案

## 获得最佳体验

### 更好个性化的技巧

1. **保持自然**：只需进行正常对话
   - 不要试图"训练" i-skill
   - 让它从您的互动中自然学习

2. **保持一致**：您的沟通风格将被更好地学习
   - 如果您偏好简洁的回答，请在请求中保持简洁
   - 如果您喜欢详细解释，请要求详细解释

3. **分享您的兴趣**：谈论您关心的话题
   - 您的爱好（体育、音乐、游戏）
   - 您的工作（编程、写作、设计）
   - 您的兴趣（科技、科学、艺术）

4. **提供反馈**：如果响应不符合您的偏好
   - 说："我更喜欢简洁的回答"
   - 说："我喜欢详细解释"
   - i-skill 将从您的反馈中学习

### 何时激活

**在以下情况下激活个性化：**
- 您与 AI 进行了 3-5 次对话
- 您想要一致的个性化响应
- 您正在使用支持个性化的技能

**无需担心：**
- 何时激活（只需说"激活个性化"）
- 如何激活（只是一个命令）
- 是否在工作（i-skill 会自动应用）

## 隐私和控制

### 您的数据，您的控制

- **本地存储**：所有数据都存储在您的设备上
- **无云端上传**：您的档案永远不会离开您的设备
- **完全控制**：您可以随时查看、编辑、删除
- **透明**：您可以准确看到 i-skill 学到了什么

### i-skill 存储的内容

i-skill 存储：
- 您兴趣的高级摘要
- 您的沟通风格偏好
- 对话中的证据（简短引用）
- 统计信息（对话次数等）

i-skill 不存储：
- 原始对话文本
- 个人信息（姓名、地址等）
- 敏感数据

## 故障排除

### 个性化不工作

**检查：**
1. 个性化是否已激活？说"查看我的档案"检查
2. 您是否有足够的对话？需要 3-5 次对话
3. 档案是否完整？查看档案以查看记录的内容

**解决方案：**
- 如果未激活，说"激活个性化"
- 如果档案不完整，进行更多对话
- 如果问题仍然存在，尝试"重置档案"并重新开始

### 档案不更新

**检查：**
1. i-skill 是否在分析对话？它应该在每次对话后更新
2. 您是否提供新信息？i-skill 仅在有新信息时更新

**解决方案：**
- 继续进行对话
- 分享新话题或偏好
- 几次对话后检查档案

### 想要重新开始

**解决方案：**
- 说"重置档案"
- 确认删除
- 开始新的对话

## 常见问题

**问：了解我需要多长时间？**
答：通常 3-5 次对话。您交谈得越多，它对您的了解就越好。

**问：我可以将 i-skill 与其他技能一起使用吗？**
答：可以！许多技能会自动激活 i-skill 以获得更好的个性化。

**问：我的数据是私密的吗？**
答：是的。所有数据都本地存储在您的设备上，从未上传。

**问：我可以编辑我的档案吗？**
答：可以。您可以随时查看和编辑您的档案。

**问：如果我不喜欢个性化怎么办？**
答：您可以随时暂停或重置个性化，或直接编辑您的档案。

- 如有必要，尝试重置并重新开始
